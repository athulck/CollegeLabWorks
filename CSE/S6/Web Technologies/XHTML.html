Syntactic Differences between HTML and XHTML

HTML XHTML
Case Sensitivity: Tags and attributes are case insensitive Tags and attributes are case sensitive and must be in lowercase
Closing Tags: Tags may or may not be closed
eg: <p> tag are not often closed 		Tags must be closed
Quoted Attribute Values: Attribute values are required to be quoted only if it contains a whitespace in between
eg: width=10 		Attribute values must be quoted
Explicit Attribute Values: Implicit attributes are not required to be stated
eg: required, disabled, checked 		Tags and attributes are case sensitive and must be in lowercase
Element Nesting: Tags may not be properly nested Tags must be properly nested
