Q) 3 Levels of Style Sheets

  1. Inline

    Highest in precedence.
    Eg: <div style="background-color: blue"> Content </div>

  2. Document Level

    Higher precedence than external, lower than inline.
    Eg: 

    		<style type="text/css">
			.red-class {
				color: red;
			}
		</style>
		<div class="red-class"> In Red</div>

  3. External Style Sheets

    Lowest in precedence. 

    		<link rel="stylesheet" type="text/css" href="styles.css">
